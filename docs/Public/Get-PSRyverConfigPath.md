# Get-PSRyverConfigPath

## SYNOPSIS
Retrieves the PSRyver config file path.

## SYNTAX

```
Get-PSRyverConfigPath [<CommonParameters>]
```

## DESCRIPTION
Retrieves the PSRyver config file path.

## EXAMPLES

### EXAMPLE 1
```
Get-PSRyverConfigPath
```

Retrieves the PSRyver config file path.
