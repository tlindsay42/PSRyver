# Test-IsWindows

## SYNOPSIS
Is this running on a Windows system?

## SYNTAX

```
Test-IsWindows [<CommonParameters>]
```

## DESCRIPTION
Is this running on a Windows system?

## EXAMPLES

### EXAMPLE 1
```
Test-IsWindows
```

Returns $true if called from a Windows system and $false if not.
